Missing parse? The scraper failed to parse the menu for this particular restaurant. This could be due to the restaurant changing the menu layout, not offering lunch for this particular week, or some other issue. Feel free to report through the Github issue tracker.

Missing restaurant? Feel free to make a pull request for any restaurant you feel is relevant.

Why? Primarily, I wanted to learn Github actions. Also, it's nice to have all menus in one place.

Who? Someone at the IT department at Axis Communications in Lund, Sweden.

How? Through a very convoluted process to generate static content using Amazon Web Services, built with SAM and hosted in S3. You can view the source code on Github. Please feel free to contribute!

Please note: This site is in no shape or form affiliated with Axis Communications AB or any of the restaurants with menus presented here.
